APDC MINI - EVENT / EVENT SEPARATE JUDGING VERSION

1. GitHub 미니 프로그램 저장소의 기존 파일을 이 압축파일 내용으로 교체합니다.
2. 모든 파일을 저장소 최상단(root)에 업로드합니다.
3. 메인(index.html)은 ENTRY SEARCH만 공개됩니다.
4. ADMIN 비밀번호: 0070
5. Dashboard는 EVENT 단위로 START / FINISH / NEXT를 운영합니다.
6. 한 EVENT 안에 여러 EVENT가 같이 플로어에서 경기할 수 있습니다.
7. Judge 화면에서는 EVENT 안의 EVENT 탭을 각각 선택하여 별도 심사합니다.
8. EVENT별로 '미제출 / 제출완료' 상태가 표시됩니다.
9. Results는 EVENT별로 독립 결과를 확정하고 CSV / 인쇄할 수 있습니다.
10. 같은 선수가 같은 EVENT 안에서 두 EVENT에 중복 배치되지 않도록 EVENT가 구성되어 있습니다.

핵심 구조:
EVENT = 실제 같은 시간에 같은 플로어에서 춤추는 경기 묶음
EVENT = 심사와 순위, 결과가 독립적으로 저장되는 실제 종목
